Temperature Sensor Simulator Python, MQTT, Grafana

Developed a temperature sensor simulator that extracts real-time data from OpenWeatherMap and sends it to a server using the MQTT protocol.

  • Used the sys library to receive input parameters and random to generate a unique 10-character identifier for each sensor.
  • Implemented an infinite loop for the continuous operation of the sensor, with detection of interruption via CTRL + C.
  • Configured periodic readings of temperature, humidity, and pressure every 10 minutes, processing the data into JSON format using json.dumps.
  • The obtained data is stored in a Python dictionary and sent to the server under the topic "datosMeteorologicos".
  • Implemented interactive visualizations in Grafana to display data on ambient temperature, relative humidity, and atmospheric pressure, with a color-coding system to identify critical conditions.

Hangman Game in C C Language

  • Developed a Hangman game using C, with the key features of the game such as:
    • Word selection.
    • Validation of player inputs.
    • Updating the game state after each turn.
  • String Handling:
    • Used character arrays (char[]) to store the hidden word and the player's guessed letters.
    • Manually implemented letter comparison and updating the correct letters in the word.
  • Control Structures:
    • Employed loops (for, while) and conditionals (if, else) to manage the game logic.
    • Controlled the number of allowed attempts and validated if the player won or lost the game.
  • Code Modularity:
    • Divided the code into functions, such as:
      • Checking if the guessed letter is in the word.
      • Updating the game progress and displaying the current state.
    • This improved code readability and maintainability.
  • Input Validation:
    • Implemented validations to handle invalid inputs (such as numerical characters) and controlled possible input errors.
  • Testing and Debugging:
    • Conducted thorough testing to ensure the game worked correctly in different scenarios.
    • Verified that the game logic correctly handled the player's victory or defeat.
